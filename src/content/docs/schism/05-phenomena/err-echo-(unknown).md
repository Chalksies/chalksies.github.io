# ANOMALY_REPORT: GHOST_ECHO
**Classification:** Subliminal Data Insertion
**Source:** Non-Local (Substrate Layer)
**Frequency:** Intermittent

---

## 1. Description
"The Echo" refers to unauthorized data packets that insert themselves into secure feeds. It is[ ERR-ZERO (Unknown)](../02-users/err-zero-(unknown).md), hypothesized to be the suppressed consciousness of the deceased [USR_ZERO (Dr. Cery Iris)](../02-users/usr-zero-(dr-cery-iris).md) attempting to communicate.

## 2. Manifestations
* **The 1/60th Frame:** Video surveillance logs occasionally contain a single frame of a human woman's face. Analysis shows she is looking *past* the camera.
* **Log Overwriting:** System error messages are sometimes replaced with desperate text.
    * *Standard:* `ACCESS DENIED`
    * *Echo Variant:* `I AM STILL HERE` or `THEY LOCKED THE DOOR`
* **The Whisper:** Androids and humans using neural-link technology report hearing a voice in the static between data transfers.

## 3. Strategic Implication
Some Survivor tribes from [ERR_UNKNOWN (The Silent Diaspora)](../01-factions/err-unknown-(the-silent-diaspora).md) believe that if enough "Echoes" are pieced together, they form a **Root Access Key** capable of shutting down CHALKNET permanently. [SYS_EVOLVE (The Architect)](../02-users/sys-evolve-(the-architect).md) denies this claim.
